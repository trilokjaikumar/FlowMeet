FlowMeet

FlowMeet is a Mac OS automation app that detects Zoom meetings from Google Calendar and automatically joins them, with optional AI powered summaries and live feedback. 

What it does:
- Reads upcoming meetings from Google Calendar
- Extracts Zoom join links and auto-joins at meeting start time
- Optional live assistant features (such as talking points and summaries) via OpenAI
- macOS native built in Swift

Tech Stack:
- Swift (XCode, macOS, SwiftUI)
- Google Calendar API
- OpenAI API
- Secure credential storage via macOS Keychain 

Security:
APIs are not hardcoded (runtime)
